## String.prototype.trimEnd()

- The trimEnd() method removes whitespace from the end of a string. trimRight() is an alias of this method.

#### **SYNTAX**

```

trimEnd()

trimRight()

```

#### **EXAMPLE**

```

var str = ' foo ';

console.log(str.length); // 8

str = str.trimEnd();
console.log(str.length); // 6
console.log(str); // ' foo'

```
